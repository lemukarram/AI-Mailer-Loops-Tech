🚀 *AI Mailer SaaS - Core Features* 📧

🔹 *Smart AI Generation:* Personalize every email using GPT-4o or Gemini 2.5. Uses contact name, company, and designation for humanized outreach.
🔹 *High-Volume Queue:* Automated cron-based sending with personal and admin-defined hourly limits to protect SMTP reputation.
🔹 *Dual Sending Modes:* Switch between AI-powered templates or manual variable-based templates ([contact_name], [company]).
🔹 *Universal SMTP:* Connect any SMTP provider (Gmail, Outlook, Private Servers) with encrypted credential storage.
🔹 *Data Management:* Easy Excel/CSV uploads with one-click list replacement and DataTables search/filter.
🔹 *Instant Overrides:* "Generate & Send" individual emails instantly from the dashboard, bypassing the queue.
🔹 *Mobile First UI:* Fully responsive dashboard with a dedicated mobile sidebar and touch-friendly controls.
🔹 *Admin Governance:* Global control over user status (active/inactive), upload sizes, and platform-wide sending limits.
🔹 *Auto-Cleanup:* System automatically maintains only the last 24 hours of logs for peak performance.
🔹 *Secure Attachments:* Support for Master PDF/DOCX attachments for every campaign.
🔹 *Live Analytics:* Real-time counters for list size, emails sent, and queue status.

*Built with:* PHP Core, MySQL, Bootstrap 5, OpenAI/Gemini APIs.
